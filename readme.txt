== Description ==

Quality Driven helps service business owners scale their businesses by reducing employee and client turnover.

=== What can Quality Driven do for you? ===
* Prevent client loss
* Reward top employees
* Simplify your quality program
* Resolve quality issues quickly to prevent poor Yelp and Google reviews
* Monitor your business from anywhere
* Simplify your performance reviews

=== What type of businesses would benefit from Quality Driven? ===
* Maid services
* Lawn care
* Window cleaning
* Carpet cleaning
* Pressure washing
* HVAC
* Electricians
* Plumbing
* And many other service businesses

Visit [Quality Driven](https://qualitydrivensoftware.com) for more information.

== Installation ==

1. Install the QDS Reviews plugin from the WordPress plugin directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the new Quality Driven section in the WordPress admin
4. Enter your QDS reviews api key, which you can find in the Reviews section of QDS
